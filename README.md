# 🎬 Movie Recommendation System

This is a simple movie recommendation system built using Python.  
It suggests movies similar to the one you select by using **content-based filtering** (similarity between movie overviews, genres, cast, etc.).

The project includes:
- A Jupyter Notebook showing how the model was made  
- A Flask app (`app.py`) to run recommendations locally  
- A trained model (`movies.pkl`)  
- TMDB dataset used for building the system  

---

## Features
- Recommends 5 similar movies based on the selected movie  
- Uses cosine similarity + movie metadata  
- Simple Flask interface  
- Easy to modify and experiment with
